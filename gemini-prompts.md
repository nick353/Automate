## 🎯 プロンプトテンプレート

### 【基本版】シンプルなWebサイト作成

```
圧倒的なクオリティで、2025年のWebデザイントレンドを踏まえたプロフェッショナルなWebサイトを作成してください。

【要件】
- テーマ: [ここにサイトのテーマを記入：例「ポートフォリオサイト」「企業サイト」「ランディングページ」]
- ターゲット: [ターゲットユーザーを記入]
- トーン: [エレガント/ポップ/ミニマル/サイバーパンクなど]

【デザイン要件】
- グラスモーフィズムやニューモーフィズムなどの最新の視覚効果を取り入れる
- パララックススクロールやスムーズなフェードイン/アウトアニメーションを実装
- レスポンシブデザインで、モバイル・タブレット・デスクトップすべてに対応
- 「このサイト、なんか好き」と無意識に感じる要素を分析し、それを実装してください

【技術要件】
- React + TypeScript + Tailwind CSS
- Framer Motion または GSAP を使用したアニメーション
- モダンなビュー・トランジション（View Transitions API）を活用

私が想像する以上の提案をしてください。Awwwards.comに掲載されるようなレベルを目指してください。
```

## 🔄 イテレーション用プロンプト

一度出力されたコードに対して、以下のような修正プロンプトを使用してください：

### デザイン修正
```
- [要素名]の位置を[方向]に調整してください
- [セクション名]の余白を[大きく/小さく]してください
- カラーパレットを[暖色系/寒色系/モノクロ]に変更してください
- フォントサイズを[大きく/小さく]してください
```

### アニメーション追加・修正
```
- [要素名]に[アニメーション名]を追加してください
- [アニメーション名]の速度を[速く/遅く]してください
- スクロール時の[要素名]の動きを[より滑らかに/よりダイナミックに]してください
- [要素名]が[箱から飛び出す/回転する/拡大する]アニメーションを追加してください
```

### 3D要素の調整
```
- 3Dカメラの初期位置を調整してください
- パーティクルの数を[増やす/減らす]してください
- Orbit Controlsの感度を調整してください
- 3Dモデルのサイズを[大きく/小さく]してください
```

### パフォーマンス最適化
```
- ページの読み込み速度を最適化してください
- アニメーションのパフォーマンスを改善してください
- 画像の最適化を行ってください
- コードをリファクタリングして可読性を向上させてください
```

## 💡 プロンプト使用のコツ

1. **段階的アプローチ**: まず基本版で全体像を作成し、その後上級版で詳細を詰める
2. **デザインガチャ活用**: 複数案を提示させ、良いとこ取りをする
3. **参考サイトの指定**: 具体的なURLやサイト名を指定すると、より明確な指示になる
4. **構造化データの保存**: 一度良い出力が出たら、その構造化データを保存して再利用
5. **イテレーション**: 完璧を目指さず、何度も修正を重ねる
