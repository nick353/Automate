# Workflow Dashboard - サービス紹介

## 🚀 Workflow Dashboard - AIが理解する、自然言語ブラウザ自動化プラットフォーム

**Workflow Dashboard**は、自然言語で指示するだけでブラウザ操作を自動化できる、次世代のワークフロー管理システムです。AIの力を借りて、複雑なブラウザ操作を簡単な言葉で実現します。

---

## ✨ 主な特徴

### 🤖 自然言語でのブラウザ自動化
プログラミングの知識は不要です。「このサイトにログインして、データを取得して、スプレッドシートに保存して」と話すように指示するだけで、AIが自動的にブラウザを操作します。

### 📺 リアルタイムライブビュー
タスクの実行中、ブラウザの動きをリアルタイムで確認できます。VNCのようなライブビュー機能により、何が起こっているかを一目で把握でき、問題があればすぐに停止・修正が可能です。

### 🎬 動画から自動生成
操作手順の動画をアップロードするだけで、AI（Gemini 1.5 Pro）が分析して自動的にタスクを作成します。チャットで不明点を確認しながら、完璧なワークフローを構築できます。

### 🔐 安全な認証情報管理
APIキーやログイン情報を暗号化して安全に保存。n8nのように、一度保存した認証情報を複数のタスクで再利用できます。PBKDF2とFernetによる強力な暗号化で、あなたの情報を守ります。

### ⏰ スケジュール実行
cron形式でタスクを自動実行。毎日、毎週、毎月、またはカスタムスケジュールで、繰り返し作業を自動化できます。

### 📊 詳細な実行履歴
すべての実行結果を記録・閲覧可能。成功・失敗のステータス、実行時間、エラーログまで、すべてを追跡できます。

### 🎯 API優先の自動化提案
AIが自動的にAPIの有無をリサーチし、可能な限りAPIを使用することを提案します。ブラウザ操作よりも効率的で確実な方法を優先します。

## 💡 なぜWorkflow Dashboardなのか？

### 従来の自動化ツールの課題
- 複雑な設定が必要
- プログラミング知識が必要
- 実行中の状態が見えない
- APIとブラウザ操作の使い分けが難しい

### Workflow Dashboardの解決策
- ✅ **自然言語で指示**: プログラミング不要
- ✅ **リアルタイム監視**: 実行中の状態を可視化
- ✅ **動画から自動生成**: 操作手順をそのまま活用
- ✅ **API優先提案**: 最適な方法を自動提案
- ✅ **安全な認証管理**: 暗号化で情報を保護
