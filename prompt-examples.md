# Gemini 3.0 Pro 実践プロンプト例集

## 🎯 すぐに使えるコピペプロンプト

### 例1: シンプルなポートフォリオサイト

```
圧倒的なクオリティで、2025年のWebデザイントレンドを踏まえたプロフェッショナルなポートフォリオサイトを作成してください。

【要件】
- テーマ: フロントエンド開発者のポートフォリオ
- ターゲット: 採用担当者、クライアント
- トーン: エレガントで洗練された、技術力が伝わるデザイン

【デザイン要件】
- グラスモーフィズムを活用したカードデザイン
- パララックススクロールとスムーズなフェードイン/アウトアニメーション
- ダークモード対応
- 「このサイト、なんか好き」と無意識に感じる要素を分析し、それを実装してください

【セクション構成】
1. ヒーローセクション（名前、肩書き、メインビジュアル）
2. About（自己紹介、スキルセット）
3. Works（プロジェクト一覧、フィルター機能付き）
4. Contact（お問い合わせフォーム）

【技術要件】
- React 18 + TypeScript + Tailwind CSS
- Framer Motion を使用したアニメーション
- レスポンシブデザイン（モバイル・タブレット・デスクトップ対応）

Awwwards.comに掲載されるようなレベルを目指してください。私が想像する以上の提案をしてください。
```

### 例2: 企業ランディングページ

```
限界を超えてください。没入感のある（Immersive）企業ランディングページを作成してください。

【要件】
- テーマ: SaaS製品のプロモーション
- ターゲット: 中小企業の経営者・IT担当者
- トーン: 信頼感があり、革新的な印象を与える

【デザイン要件】
- ニューモーフィズムとグラスモーフィズムを組み合わせた視覚効果
- スクロールに連動したアニメーション（要素が順番にフェードイン）
- ビュー・トランジションによる滑らかなページ遷移
- マイクロインタラクション（ボタンホバー、フォーム入力など）

【セクション構成】
1. ヒーローセクション（製品の価値提案、CTAボタン）
2. 特徴セクション（3つの主要機能、アイコン付き）
3. 社会証明セクション（顧客の声、導入企業ロゴ）
4. 料金プランセクション（比較表）
5. CTAセクション（最終的なコンバージョンポイント）

【技術要件】
- React + TypeScript + Tailwind CSS
- GSAP を使用した高度なアニメーション
- フォームバリデーション機能
- アナリティクス統合準備

【品質基準】
- Lighthouse Score 90以上
- モバイルファーストのレスポンシブデザイン
- WCAG 2.1 AA 準拠

Behanceで高評価を受けるサイトを参考に、コンバージョンを最大化するデザインを実現してください。
```

### 例3: 3Dインタラクティブサイト

```
圧倒的なクオリティで、React Three Fiberを使用した没入感のある3D Webサイトを作成してください。

【コンセプト】
インタラクティブな3D体験を提供する、クリエイティブエージェンシーのポートフォリオサイト

【必須技術】
- React Three Fiber (@react-three/fiber)
- @react-three/drei（ヘルパー関数）
- GSAP（アニメーション制御）
- Orbit Controls（マウス操作によるカメラ制御）
- パーティクルシステム

【デザイン要件】
- グラスモーフィズムとノイズエフェクトを組み合わせたUI
- スクロールに連動した3Dカメラアニメーション
- マウスホバーで反応するインタラクティブな3Dオブジェクト
- パーティクルエフェクトによる没入感の演出

【セクション構成】
1. 3Dヒーローセクション（回転する3Dオブジェクト、パーティクル）
2. About（スクロールで3Dシーンが変化）
3. Works（3Dギャラリー、Orbit Controlsで操作可能）
4. Contact（フォーム）

【パフォーマンス要件】
- 60fpsを維持する最適化
- LOD（Level of Detail）の実装
- WebGLの最適化

【参考】
Awwwards.comの「3D」カテゴリで受賞したサイトを参考にしてください。

私が想像する以上の提案をしてください。限界を超えてください。
```

### 例4: デザインガチャ版（3案提示）

```
限界を超えてください。没入感のあるWebサイトを作成してください。

【テーマ】
フリーランスデザイナーのポートフォリオサイト

【デザインガチャ】
異なるアプローチのデザイン案を3パターン提示してください：

1. **ミニマル・エレガント系**
   - 高級感のある、余白を活かしたデザイン
   - モノクロ + 1つのアクセントカラー
   - シンプルなアニメーション

2. **ポップ・カラフル系**
   - エネルギッシュで目を引くデザイン
   - グラデーションとビビッドなカラー
   - ダイナミックなアニメーション

3. **サイバーパンク・フューチャリスティック系**
   - 3D要素やパーティクルを活用
   - ネオンカラーとダークテーマ
   - インタラクティブな3D体験

各案について以下を提示してください：
- デザインコンセプトの詳細説明
- カラーパレット（HEXコード付き）
- 主要なアニメーション効果の説明
- 使用する技術スタック
- コードの実装例（主要セクションのみ）

【必須要素】
- グラスモーフィズム、ニューモーフィズム、ノイズエフェクト
- パララックス、スクロールアニメーション、ビュー・トランジション
- React Three Fiberを使用した3D要素（案3のみ必須、他はオプション）

【品質基準】
Awwwards.comやBehanceで高評価を受けるサイトを参考に、プロフェッショナルなクオリティを実現してください。
```

## 🔧 修正・イテレーション用プロンプト例

### デザイン調整
```
以下の修正をお願いします：
- ヒーローセクションのタイトルを中央揃えから左揃えに変更
- カラーパレットのプライマリカラーを #6366f1 から #8b5cf6 に変更
- 全体的な余白を20%大きくしてください
- フォントサイズを1.2倍に拡大してください
```

### アニメーション追加
```
以下のアニメーションを追加してください：
- 作品カードにホバー時に「箱から飛び出す」ような3D効果
- スクロール時に要素が順番にフェードインするアニメーション
- ボタンにマイクロインタラクション（ホバー時の拡大、クリック時のリップル効果）
- ページ遷移時にビュー・トランジションを適用
```

### 3D要素の調整
```
3D要素について以下の調整をお願いします：
- パーティクルの数を500から1000に増やしてください
- Orbit Controlsの感度を下げて、より滑らかな操作感にしてください
- 3Dカメラの初期位置を調整して、より良いアングルにしてください
- パフォーマンスを最適化して、60fpsを維持してください
```

### パフォーマンス改善
```
以下の最適化をお願いします：
- 画像をWebP形式に変換し、遅延読み込みを実装
- アニメーションのパフォーマンスを改善（will-changeプロパティの適切な使用）
- コードをリファクタリングして、バンドルサイズを削減
- Lighthouse Scoreを90以上に改善
```

## 💡 プロンプト組み合わせのコツ

### 1. 段階的アプローチ
```
ステップ1: まず基本版で全体像を作成
ステップ2: デザインガチャで方向性を決定
ステップ3: 構造化プロンプトで詳細を実装
ステップ4: イテレーションで微調整
```

### 2. 参考サイトの具体的な指定
```
以下のサイトを参考にしてください：
- [Awwwards - Site of the Day](URL)
- [Behance - Featured Project](URL)
- [Dribbble - Shot](URL)

特に以下の要素を取り入れてください：
- [サイト1]のカラーパレット
- [サイト2]のアニメーション効果
- [サイト3]のレイアウト構造
```

```
【制約】
- [ライブラリ名]は使用しないでください

- [機能名]は実装しないでください
