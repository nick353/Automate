# Gemini 3.0 Pro プロンプト作成 クイックリファレンス

## 🎯 基本テンプレート（コピペ用）

```
圧倒的なクオリティで、2025年のWebデザイントレンドを踏まえたプロフェッショナルなWebサイトを作成してください。

【要件】
- テーマ: [サイトのテーマ]
- ターゲット: [ターゲットユーザー]
- トーン: [エレガント/ポップ/ミニマルなど]

【デザイン要件】
- グラスモーフィズムやニューモーフィズムなどの最新の視覚効果
- パララックススクロールやスムーズなアニメーション
- レスポンシブデザイン対応
- 「このサイト、なんか好き」と感じる要素を実装

【技術要件】
- React + TypeScript + Tailwind CSS
- Framer Motion または GSAP
- モダンなビュー・トランジション

Awwwards.comに掲載されるようなレベルを目指してください。私が想像する以上の提案をしてください。
```

## 🔑 必須キーワード

### パワー形容詞（品質を底上げ）
- 圧倒的なクオリティで
- 限界を超えてください
- エレガントで高級感のある
- 没入感のある（Immersive）
- プロフェッショナルな
- 2025年のWebデザイントレンドを踏まえて

### 専門用語（リッチな表現）
**視覚効果:**
- Glassmorphism（グラスモーフィズム）
- Neumorphism（ニューモーフィズム）
- Noise Effect（ノイズエフェクト）
- Gradient Mesh（グラデーションメッシュ）

**アニメーション:**
- Parallax Scrolling（パララックス）
- Scroll-triggered Animations（スクロールアニメーション）
- Fade In/Out（フェードイン/アウト）
- View Transitions API（ビュー・トランジション）
- Micro-interactions（マイクロインタラクション）

**3D要素:**
- React Three Fiber
- Orbit Controls（カメラ操作）
- Particle System（パーティクル）

### 創造性を刺激するフレーズ
- 「このサイト、なんか好き」と無意識に感じる要素を分析し、それを実装してください
- 私が想像する以上の提案をしてください
- 余白を残してAIの創造性に任せる

### 品質基準
- Awwwards.comに掲載されるようなレベルで
- Behanceで高評価を受けるサイトを参考に
- Dribbbleのトップデザインを参考に

## 📋 セクション構成の例

### ポートフォリオサイト
1. ヒーローセクション（名前、肩書き、CTA）
2. About（自己紹介、スキル）
3. Works（作品ギャラリー）
4. Contact（お問い合わせ）

### ランディングページ
1. ヒーローセクション（価値提案、CTA）
2. 特徴セクション（主要機能）
3. 社会証明（顧客の声、ロゴ）
4. 料金プラン
5. CTAセクション

## 🎨 デザインスタイルの指定

### ミニマル・エレガント
- 高級感のある、余白を活かしたデザイン
- モノクロ + 1つのアクセントカラー
- シンプルなアニメーション

### ポップ・カラフル
- エネルギッシュで目を引くデザイン
- グラデーションとビビッドなカラー
- ダイナミックなアニメーション

### サイバーパンク・フューチャリスティック
- 3D要素やパーティクルを活用
- ネオンカラーとダークテーマ
- インタラクティブな3D体験
