# Gemini 3.0 Pro 高品質Webサイト作成プロンプト集

このリポジトリには、Gemini 3.0 Pro（Google AI Studio）を使ってプロ級の高品質なWebサイトやアプリを作成するためのプロンプトテンプレートと実践例が含まれています。

## 📁 ファイル構成

- **`gemini-prompts.md`** - プロンプトテンプレート集（基本版、中級版、上級版など）
- **`prompt-examples.md`** - すぐに使える実践的なプロンプト例
- **`README.md`** - このファイル（使い方ガイド）

## 🚀 クイックスタート

### 1. 基本の使い方

1. `prompt-examples.md`から、あなたの目的に合ったプロンプト例を選ぶ
2. `[ ]`内の部分を自分の要件に合わせてカスタマイズ
3. Gemini 3.0 Pro（Google AI Studio）に貼り付けて実行

### 2. 段階的なアプローチ

```
ステップ1: 基本版で全体像を作成
  ↓
ステップ2: デザインガチャで方向性を決定（複数案から選択）
  ↓
ステップ3: 構造化プロンプトで詳細を実装
  ↓
ステップ4: イテレーションで微調整
```

## 📚 プロンプトの種類

### 基本版
シンプルなWebサイト作成に最適。初心者でも使いやすい。

### 中級版（デザインガチャ）
複数のデザイン案を提示させ、良いとこ取りができる。

### 上級版（構造化プロンプト）
JSON形式で詳細を指定。再現性が高く、複雑なサイトに最適。

### 特化版
- 3Dサイト特化版
- ランディングページ特化版
- ポートフォリオサイト特化版

## 💡 使用のコツ

### カスタマイズ方法
1. **テーマの変更**: `[テーマ]`部分を自分の目的に合わせて変更
2. **技術スタックの指定**: 使用したい/したくない技術を明確に指定
3. **参考サイトの追加**: 具体的なURLを指定するとより明確な指示になる

### イテレーション
一度の出力で完璧を目指さず、以下のような修正プロンプトを繰り返し使用：
- 「[要素名]の位置を調整してください」
- 「[アニメーション名]を追加してください」
- 「もっと余白を大きくしてください」

### デザインガチャの活用
複数案を提示させ、良いとこ取りをする：
- 案1のカラーパレット
- 案2のアニメーション効果
- 案3のレイアウト構造
